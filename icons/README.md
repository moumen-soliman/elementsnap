# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Generating Icons

1. **Using the HTML generator:**
   - Open `create-icons.html` in your browser
   - Click "Generate Icons"
   - Right-click each canvas and save as the corresponding PNG file in this directory

2. **Manual creation:**
   - Create icons with a design tool (Figma, Sketch, etc.)
   - Export as PNG files with the exact dimensions listed above

3. **Online tools:**
   - Use an online icon generator
   - Ensure icons are square and match the required dimensions

## Icon Design Tips

- Use a simple, recognizable design (e.g., "E" for ElementSnap, or a selection/cursor icon)
- Ensure icons are readable at small sizes (16x16)
- Use high contrast for visibility
- Consider both light and dark backgrounds
